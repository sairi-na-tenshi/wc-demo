Web Component

Фреймворк для модульной организации клиентского кода. Скрипты, стили, картинки и другие сопутствующие файлы группируются в директории создавая автономные сущности, которые легко добавлять, удалять и переносить между проектами. На стадии разработки они подключаются по отдельности, позволяя легко локализовывать места ошибок, но для продакшена создаются специальные скомпилированные версии, минимизирующие число запросов к серверу.
